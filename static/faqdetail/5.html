Minimumskrav til TrustBox

  • Windows:
    1. 2003 eller senere (XP, Vista, 7 og 8), og Server 2003-2008.
    2. 1 GHz Intel eller tilsvarende CPU.
    3. 512 MB RAM.
    4. Bredbånds internetforbindelse.
    5. Internet Explorer 7 eller nyere / Firefox 2.0 eller nyere / Google Chrome / Safari.
    6. Den nyeste version af Java og Flash.
  • Mac:
    1. OS X 10.6.8 eller nyere.
    2. 1 GHz Intel eller tilsvarende CPU.
    3. 512 MB RAM
    4. Bredbånds internetforbindelse.
    5. Internet Explorer 7 eller nyere / Firefox 2.0 eller nyere / Google Chrome / Safari.
    6. Den nyeste version af Java og Flash.
